Operation: Ethreal Web Security Assessment

Agent, your mission is to assess the security of Ethreal, a fictitious entity operating under unknown directives. Your objective is to conduct a vulnerability assessment using Wapiti and Nikto, analyze the reports, and recommend security improvements.

Mission Objectives

1. Infiltrate the Target Network

  • Access the Ethreal website running on https://ethreal.com.
  • Confirm successful connectivity and reconnaissance parameters.

2. Deploy Web Vulnerability Scanners

  • Execute a series of Wapiti and Nikto scans against the target.
  • Generate structured HTML reports for review.

3. Analyze Collected Data

  • Compare vulnerabilities detected by Wapiti and Nikto.
  • Evaluate the differences in scanning methodologies and detection capabilities.

4. File Your Intelligence Report

  • Submit both HTML reports.
  • Provide a detailed analysis of findings.
  • Recommend defensive countermeasures Ethreal could implement to mitigate security risks.

Mission Execution: Wapiti & Nikto Commands

Initialize the Lab

labtainer webscan

Visit the Target Site (Optional)

firefox https://ethreal.com

Run Wapiti Scan

Execute Wapiti against the target web application and generate an HTML report.

wapiti -u http://ethreal.com -o wapiti_report -f html

Run Nikto Scan

Run Nikto to scan the same target and output an HTML report.

  1. wget --spider --no-check-certificate -r -nd https://ethereal.com -o urls.txt
  2. cat urls.txt | grep "https" | awk '{print $3}' | sort -u > targets.txt
  3. for url in $(cat targets.txt); do nikto -h $url -o "nikto_${url##*/}.html" -Format htm; done

Intelligence Report: Analysis Questions

  • Which tool detected more vulnerabilities? Why do you think that is?
  • What types of vulnerabilities did Wapiti detect that Nikto missed?
  • What types of vulnerabilities did Nikto detect that Wapiti missed?
  • How do the scanning methodologies differ between Wapiti and Nikto?
  • What defensive measures could mitigate these vulnerabilities?

Final Submission

Deliverables

  • Wapiti Report (HTML file).
  • Nikto Report (HTML file).
  • Mission Analysis Report answering the questions above.
